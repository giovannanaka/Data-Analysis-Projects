# Automated File Sorter

## Overview
This project demonstrates a Python automation script that organizes files
based on their extension.

The script:
- Detects CSV and Excel files
- Creates folders if they do not exist
- Moves files into the appropriate directories

## Tools Used
- Python
- os
- shutil
- pandas
- matplotlib

## Example Workflow
Messy folder:
customers.xlsx
inventory.xlsx
data_2023.csv
data_2024.csv

After running the script:
CSV Files/
Excel Files/

## Purpose
This automation demonstrates how repetitive file organization tasks can
be handled efficiently with Python.
